How to Install

Prerequisites

  • A PS3 running CFW/HEN
  • A PC and a Flash Drive/PS3 FTP access (unless using webman and the ONE-CLICK install)

Using a PC Install or you can use the auto-installer which is easier on the PS3 with Webman Mod

  1. Transfer PKG onto the root of the USB/SD Card
  2. Go to Install Packages -> standard -> Install the file you just downloaded

Using Discord CDN

  1. Click on file put it on your USB/SD card or if you have patches install it to /dev_hdd0/packages
  2. Go to the game tab and click on Package Manager and Install Package Files
  3. if you have it on a USB/SD Card go to standard and if you installed it to /dev_hdd0/packages select PS3 System Storage
